# VisualDon - Un été en Suisse 
Sitw web : Visualisation interactive des festivals de musique suisses durant l'été 2025.
 
## Contexte
Les données viennent des sites officiels des festivals (Paléo, Street Parade, 
Montreux Jazz, etc.), de Wikipedia et de plateformes comme Ticketcorner. 
Ce sont les organisateurs eux-mêmes qui publient ces infos, principalement 
pour vendre des billets et promouvoir leurs événements.
 
Quelques biais à noter : les chiffres de fréquentation sont des estimations 
des organisateurs, donc probablement arrondis à la hausse. Les petits festivals 
comme Caribana ou Venoge sont aussi beaucoup moins documentés que les grands 
comme Paléo ou la Street Parade.
 
## Description des données
On a collecté manuellement les données suivantes pour chaque festival :
- Nom, ville, canton, coordonnées GPS
- Dates et durée
- Genre musical
- Fréquentation estimée
- Prix des billets (min/max)
- Artistes têtes d'affiche
 
Les données sont stockées dans un fichier JSON et un CSV.
 
## But
On veut raconter l'été suisse comme un voyage. L'idée c'est de montrer 
qu'il se passe quelque chose chaque week-end, dans toute la Suisse, 
pour tous les goûts du jazz au hip-hop en passant par la techno.
 
L'approche est à la fois exploratoire (découvrir la carte des festivals) 
et explicative (guider le lecteur dans un récit chronologique de juin à août).
